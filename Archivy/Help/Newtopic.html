Общие сведения об Archive


Archive это архиватор для Windows — инструмент для архивирования и управления архивами.


Archive для Windows поставляется в одном варианте:


  •    версия с графическим интерфейсом пользователя (GUI) - Archive.exe;


Archive работают в среде Windows.


Некоторые отличительные особенности Archive:


  •    полная поддержка архивов ZIP;
  •    графический интерфейс с поддержкой перетаскивания (drag & drop);
  •    при архивации документов применяется отдельный метод сжатия данных;
  •    работа с документами других форматов (.xml .html .txt .rtf .doc. docx. xls .xlsx);
  •    после сжатия документов нашим методом мы получаем сжатую версию. Такие файлы при распаковки декодируются обратно в первоначальное состояния;
  •    поддержка кодировки Юникод в именах файлов;
