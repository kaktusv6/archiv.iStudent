Режимы графического интерфейса


Чтобы использовать Archive в режиме графической оболочки, дважды щёлкните левой кнопкой мыши на значке Archive — после этого для архивации и извлечения файлов вы сможете пользоваться кнопками и меню.



Как заархивировать файлы с помощью графической оболочки Archive


Сначала нужно открыть архив в Archive нажмите кнопку "Открыть архив" вверху окна Archive, выберите в появившемся окне нужный архив, после чего нажмите кнопку OK.


Теперь нужно добавить файлы. Это можно сделать двумя способами:


  • нажав в пункте меню "Добавить файлы"
  • перетащить архив на окно Archive. Перед тем как это сделать, убедитесь, что в окне Archive не открыт другой архив.



Как распаковать архив с помощью графической оболочки Archive


Теперь нажмите кнопку "Распаковать архив" вверху окна Archive, введите в появившемся окне нужный путь (папку), после чего нажмите кнопку OK.



Как создать архив с помощью графической оболочки Archive


нажав кнопку "Создать архив"  вверху окна Archive, выберите в появившемся окне нужную папку где будет сохранен новый пустой архив (.zip). Затем в этот архив можно добавлять файлы (см.выше)
